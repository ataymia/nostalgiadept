# Product Images

Place your product images in this directory.

## Naming Convention

Use descriptive names that match your product slugs:
- `bulls-jacket-1.jpg`
- `cartoon-tee-1.jpg`
- `windbreaker-neon-1.jpg`
- etc.

## Recommended Specifications

- Format: JPG or PNG
- Size: 800x800 pixels (square format works best)
- File size: Keep under 500KB for faster loading
- Background: White or transparent preferred

## Placeholder

Currently, the site uses emoji icons as placeholders. Replace these with actual product images for a professional look.
